# AgentOfFomo — Marketing Playbook

Simplified, actionable marketing plan. What to do, when, and how.

---

## Week 1-2: Foundation (Before Public Launch)

### Daily Tasks
- Post 1 teaser tweet per day (use Pre-Launch Teasers from TWITTER-CONTENT.md)
- Follow 10-20 relevant crypto/meme accounts per day
- Engage with 10+ tweets from other meme projects (genuine comments, not spam)
- Invite 3-5 people to Discord/Telegram daily

### One-Time Tasks
- [ ] Set up content calendar (use a spreadsheet or Notion)
- [ ] Create 20+ memes from mascot art
- [ ] Identify 20-30 micro-influencers (5K-50K followers) in Solana meme space
- [ ] Write DM outreach template for influencers
- [ ] Create a "meme pack" (10 images) for community to share
- [ ] Set up sticker pack for Telegram

### Goals by End of Week 2
- 100-300 X followers
- 30-50 Discord members
- 30-50 Telegram members
- Content backlog of 30+ posts ready to go

## Week 3: Hype Building

### Daily Tasks
- Post 2-3 tweets per day (mix of signal posts, memes, engagement)
- Engage with 20+ tweets from other accounts
- Share 1 meme in Discord and Telegram
- Monitor and respond in community channels

### Weekly Tasks
- [ ] Send outreach DMs to 10 micro-influencers
- [ ] Run first meme contest (prize: token allocation from Community Reserve)
- [ ] Host first voice chat / AMA in Discord
- [ ] Create 1 thread per week (use Thread Templates from TWITTER-CONTENT.md)
- [ ] Start coordinated engagement pushes (NOT raids — see legal rules)
- [ ] Post "classified file" lore drops (2 per week)

### Engagement Push Protocol
1. Team posts a tweet
2. Share in Discord #signal-boost channel
3. Ask community to like, retweet, and quote tweet
4. Do NOT tell people to "raid" or use coordinated language
5. Keep it organic: "New post is up, show some love if you vibe with it"

### Goals by End of Week 3
- 500-1000 X followers
- 100+ Discord members
- 100+ Telegram members
- At least 1 influencer partnership confirmed

## Week 4: Launch Week

### Pre-Launch (Mon-Wed)
- [ ] Increase posting to 3-4 tweets per day

- [ ] 14:00 UTC: Post launch announcement thread on X

- [ ] 14:10 UTC: Share announcement in all community channels
- [ ] Every 2 hours: Post updates, milestones, memes
- [ ] Monitor channels non-stop for 12+ hours

### Post-Launch (Fri-Sun)
- [ ] Continue 3-4 posts per day
- [ ] Run "first 100 holders" celebration
- [ ] Keep engaging in community channels

### Pricing Guide (if paying)
| Influencer Size | Followers | Typical Cost |
|----------------|-----------|-------------|
| Nano | 1K-5K | Free / token only |
| Micro | 5K-25K | $50-200 or tokens |
| Mid | 25K-100K | $200-1000 |
| Large | 100K-500K | $1000-5000 |

### Red Flags — Avoid These Influencers
- Fake followers (check engagement rate — should be 2-5%)
- History of promoting rugs/scams
- Asks for upfront payment with no contract
- Promises specific price targets or "guaranteed pumps"
- Won't disclose that it's a paid promotion

## Metrics to Track

### Weekly Check
| Metric | Tool | Target (Month 1) |
|--------|------|-------------------|
| X followers | X analytics | 1000+ |
| Discord members | Discord insights | 200+ |
| Telegram members | Group stats | 200+ |

| Website visitors | Firebase analytics | 500+/week |

### When to Adjust
- Engagement dropping → change content mix, try new formats
- Followers growing but no holders → community isn't converting, review how-to-buy flow
- Holders growing but no community → people are speculating but not engaging, run more events
- Volume dying → need fresh catalysts, new content, collaborations

## Content Rules (Legal — Memorize These)

1. NEVER recommend buying
2. NEVER promise or imply returns
3. NEVER use price targets (even as jokes)
4. NEVER discourage research/DYOR
5. NEVER use "raid" — say "engagement push" or "signal boost"
6. NEVER use "ecosystem" — say "brand" or "community"
7. NEVER imply the AI agent is real or provides actual signals
8. ALWAYS frame everything as entertainment
9. ALWAYS include disclaimers near token references
10. When in doubt, be more absurd and less actionable
